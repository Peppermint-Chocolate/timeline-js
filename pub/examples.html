Examples of Timeline.JS

Timeline.JS Examples Documentation

I have constructed our semester timeline using Timeline.JS. You can have some fun playing with it, or scroll down some more to see how it was made!

1. Create a New Timeline

Firstly, we should make sure in the HTML file's there exists a div tag with a unique ID.

< div id="Example_1">< /div>

We could initialize the new timeline from passing our expected timeline ID, timeline title, width and background color as arguments to the constructor. The first argument should match the ID in div tag where the timeline will be placed.

const t1 = new TimelineGenerator()

t1.makeTimeline("Example_1", "Example1", "CSC309H1 Semester Timeline", "1000px", "rgba(163, 161, 165, 0.7)")

2. Add Events into the Timeline

We can create an event by specifying its unique event ID, title, content, position on timeline, as well as the background color. The first argument should still match the div tag of the timeline.

We could call makeEvent multiple times to add many events to the timeline.

t1.makeEvent("Example_1", "Start_2", "Start of Term", "Course starts", "50px")

t1.makeEvent("Example_1", "End_2", "End of Term", "Course ends", "950px")

t1.makeEvent("Example_1", "A_1_2", "Asgmt.1", "Ex 1", "150px")

///// try the code below yourself! ////

t1.makeEvent("Example_1", "A_2_2", "Asgmt.2", "Exercise on library system", "390px")

// Hint: Event Id = "A_2_2", Title = "Asgmt.2", etc.

t1.makeEvent("Example_1", "A_3_2", "Asgmt.3", "Eexercise on restaurant reservation", "570px")

3. Update Title/Content of an Event

We could update the title and/or content of an event by specifying its unique Event ID. The function automatically updates the event title and/or content based on what you input, and the rest of the event remains the same.

//// try the code below yourself! ////

t1.updateEventContent("A_1_3", "", "Exercise on twitter layout")

// Hint: Event Id = "A_1_3", Content = "Exercise on twitter layout", etc.

4. Update Position of an Event

We could update the position of an event by specifying its unique Event ID. The input should be in the range of 0px and the timeline's width in px.

///// try the code below yourself! ////

t1.updateEventPosition("A_2_4", "260px")

// Hint: Event Id = "A_2_4", Position = "260px"

5. Create a New Tag

We could create tags to categorize the events. All tags are automatically displayed at the bottom left of the timeline. You could also choose a color for your new tag. Tag_name should not contain any spaces. For asthetic reasons, a maximum of 3 tags are displayed.

t1.createTag("Assignments", "rgba(250, 255, 229, 0.8)") // add a yellow tag

t1.createTag("Team_Project", "rgba(249, 231, 241, 0.8)") // add a pink tag

///// try the code below yourself! ////

t1.createTag("Individual_Project", "rgba(222, 245, 244, 0.8)") // add a blue tag

// Hint: Tag_Name = "Individual_Project", pick a color you like

6. Attach a Tag to an Event

We could add an existing tag to an event by specifying the Event ID and the tag_name. The target event will be automatically appear to be in the tag's colour. The event title will also appear to the right of the specified tag at the bottom left of the timeline.

t1.addTagToEvent("A_1_6", "Assignments")

///// try the code below yourself! ////

t1.addTagToEvent("A_2_6", "Assignments")

// Hint: Event ID = "A_2_6", Tag_Name = "Assignments"

t1.addTagToEvent("A_3_6", "Assignments")

From repeating the previous steps, we could add, update, and categorize a few more events. We can now obtain a timeline similar to the one below. Note that the "Start of Term" and "End of Term" events does not belong to any tag. Their colour is not specified so they are still displayed in the default/base event color (coconut white).

7. Remove a Tag

We could remove a tag from the timeline by specifying its tag_name. Deleting a tag will automatically remove the tag name and its events list from the bottom left of the timeline. All events previously belonged to this tag will now be displayed in the default/base color.

t1.removeTag("Team_Project") // remove the pink tag

///// try the code below yourself! ////

t1.removeTag("Assignments") // remove the yellow tag

// Hint: Tag_Name = "Assignments"

8. Delete an Event

We could delete an event from specifying its unique Event ID. The event will then be removed from the timeline. If the event belongs to a tag, the event's title will also be removed from the tag's event list at the bottom left of the timeline.

t1.deleteEvent("IP_1_8") // delete "Individual Proposal"

///// try the code below yourself! ////

t1.deleteEvent("A_1_8") // delete "Asgmt.1"

// Hint: Event ID = "A_1_8"

t1.deleteEvent("IP_2_8") // delete "Individual Project"

9. Change Color for Base (untagged) Events

We could specify a default/base colour for the events that don't belong to any tag. All events satifying this criteria will be coloured automatically.

///// try the code below yourself! ////

t1.changeColorForBaseEvents("rgba(214, 247, 197, 1)") // change to bright green

// Hint: Pick a color you like!

10. Change Timeline Background Color

We could specify a colour to be applied as the background colour of the timeline.

///// try the code below yourself! ////

t1.changeBackground("rgba(123, 177, 234, 0.7)"); // change to blue

// Hint: Pick a color you like!

11. Clear All Events from Timeline

We could remove all existing events and tags from the timeline. Unless specified, the base colour will remain the same when creating new events to the now empty timeline.

///// try the code below yourself! ////

t1.clearTimeline();

// Hint: Simply click the button!
