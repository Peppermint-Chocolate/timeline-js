Documentation of Timeline.JS

Timeline.JS Examples Documentation

Getting Started

To get started, you should import the Timeline.JS library as s script shown below into the Head of your html file.
<script defer type="text/javascript" src='js/Timeline.js'></script>
Before creating a new timeline, make sure that in the Body of your html file you have a Div tag that has a unique Id. This Id should be identical to the timelineId of the timeline you will construct. An example is given below:
<div id="Example_1"></div>

JS Object Structure

Property Description Example Value
tId (string) The unique ID of the timeline "Timeline_1"
eId (string) The unique ID of the event "Event_1"
tTitle (string) The title of the timeline "CSC309H1 Semester Timeline"
tBackgroundColor (string) Colour code that defines background color of the timeline "rgba(95, 95, 95, 0.2)"
width (string) The width of the timeline in px "800px"
eTitle (string) The title of the event "Alpha Release"
content (string) The content of the event "Exercise on twitter layout"
posX (string) The position of of the event on the timeline "260px"
eBackgroundColor (string) The colour code that defines background color of the event "rgba(95, 95, 95, 0.2)"
events (dictionary) Pairs of event Id and tag name {{Start: "default"}, {A_1: "Assignments"}, {P_1: "Team_Project"}}
tags (dictionary) Pairs of tag name and tag color code {{default: "rgba(255, 255, 255, 0.8)"}, {Assignments: "rgba(250, 255, 229, 0.8)"}, {Team_Project: "rgba(249, 231, 241, 0.8)"}}

Common Methods

Constuctor of Timeline

This function constructs a new instance of timeline.

// example

const t0 = new TimelineGenerator();

makeTimeline(querySelector, tId, title, width = "1000px", backgroundColor = "rgba(50, 50, 50, 0.3)")

This function creates a new timeline in the Div tag specified by the input. Unless specified, the timeline will have a default width of 1000px and a background color of light grey.

// example

t0.makeTimeline("Example_0", "Example0", "CSC309H1 Semester Timeline", "1000px", "rgba(163, 161, 165, 0.7)");

makeEvent(querySelector, eId, title, content = "", posX = "0px")

This function adds a new event to the timeline by inputting unique event Id, title, content, and position. If the content does not have any input, an event with empty content will be created.

// example

t0.makeEvent("Example_0", "A_2", "Asgmt.2", "Exercise on library system", "390px");

updateEventContent(eId, title = "", content = "")

This fucntion updates the title and/or content of a event identified by the input event Id. For example, if the user didn't input anything for the event title, the original event title will be displayed.

// example

t0.updateEventContent("A_1", "", "Exercise on twitter layout");

updateEventPosition(eId, posX)

This fucntion updates the position of a event identified by the input event Id. The posX should be a number between 0px and the width of the timeline.

// example

t0.updateEventPosition("A_2", "260px");

createTag(tagName, tagColor = "rgba(255, 255, 255, 0.8)")

This function creates a tag with name and color specified by the input. The tag name cannot contain spaces. If no color is specified, the tag color will be initialized as default (coconut white). A maximum of 3 tags will be displayed on the timeline due to space restriction.

// example

// create a "Individual_Project" tag and initialize the tag color to blue

t0.createTag("Individual_Project", "rgba(222, 245, 244, 0.8)");

addTagToEvent(eId, tagName)

This function addes a tag specified by its tagName to an event specified by its event Id. The event will now have the same color as the tag. The change can also be seen at the bottom left of the timeline, where a list of events are displayed.

// example

t0.addTagToEvent("IP_1", "Individual_Project");

changeColorForBaseEvents(backgroundColor = "rgba(255, 255, 255, 0.8)")

This function takes in a color value as input and changes the color of all untagged events accordingly. Newly created events will also display the changed base colour. Events that belongs to a tag will remain its original color.

// example

// change base color to light green

t0.changeColorForBaseEvents("rgba(214, 247, 197, 1)");

changeBackground(backgroundColor)

This function takes in a color value as input and change the background color of the timeline accordingly.

// example

// change the base color to blue

t0.changeBackground("rgba(123, 177, 234, 0.7)");

removeTag(tagName)

This function removes a tag as specified by its input tag name. The tag will be removed from the list of tags displayed at the bottom left of the timeline. All events that belongs to this tag will also displayed the base/default color.

// example

// remove the "Team_Project" tag

t0.removeTag("Team_Project");

deleteEvent(eId)

This function deletes an event from the timeline as specified by its input event Id. The event will be removed from the timeline display. If the event belongs to a tag, the event title will also be removed from the list of tags at the bottom left of the timeline.

// example

t0.deleteEvent("A_2");

clearTimeline(backgroundColor = "rgba(50, 50, 50, 0.3)")

This function clears all events and tags from the timeline, and reset its background color to default light grey.

// example

t0.clearTimeline();
